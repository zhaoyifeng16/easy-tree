# easy-tree
常用的树工具

```
$ npm install @zhaoyifeng/easy-tree
```
```javascript
import * as easyTree from '@zhaoyifeng/easy-tree'
```
```javascript
import { flat, filterNode } from '@zhaoyifeng/easy-tree'
```

## API
### flat
扁平化

### findNode
查找

### filterNode
筛选

### insertBefore
向节点前插入数据

### insertAfter
向节点后插入数据

### insertToChild
插入到子节点

### updateNode
修改节点

### deleteNode
删除节点
